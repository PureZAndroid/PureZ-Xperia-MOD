Magisk Module/Flahable : PureZ Xperia MOD

PureZ Xperia MOD is Xperia HOME + Theme Icons + Sony Xperia Apps

- Android 5.0 and Up
- Magisk 14.0 and Up
- TWRP or Magisk Manager App

How To Install:
- Donwnload PureZ Xperia MOD Zip
- Flash Zip form TWRP
