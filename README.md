# PureZ Xperia MOD

### PureZ Xperia MOD (Magisk Module) : Xperia Home + ThemeIcons + Sony Xperia Apps

### Requirements:
- Android 5.0 and Up
- Magisk 15.0 and Up
- TWRP Recovery or Magisk Manager

### How To Install:
- Donwnload PureZ Xperia MOD Zip
- Flash Zip form TWRP
- Reboot
- Done

(OR)

- Open Magisk Manager
- Go to Modules
- Click Add Button
- Choose PureZ Xperia MOD Zip
- Installaion time...
- Then,You need to Reboot
- Done
